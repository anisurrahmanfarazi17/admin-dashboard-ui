# Admin Dashboard UI

A modern and responsive admin dashboard interface built using HTML and CSS.

Designed to showcase dashboard layouts, analytics cards, tables, sidebar navigation, and responsive user interfaces.

## 🚀 Features

- Responsive Dashboard Layout
- Sidebar Navigation
- Analytics Cards
- Statistics Overview
- Recent Activity Table
- User Profile Widget
- Dark Theme Design
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
